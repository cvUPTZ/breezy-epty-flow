
# Application Description

This web application is a comprehensive platform for football (soccer) match analysis and management. It provides a complete ecosystem for recording, analyzing, and visualizing match data, with advanced collaboration features and administrative tools for coaches, analysts, and football organizations.

## Key Features

### Match Data Management & Event Tracking
* **Real-time Event Recording:** Users can meticulously record match events in real-time, including goals, assists, fouls, substitutions, and detailed player actions
* **Advanced Ball Tracking:** The system supports sophisticated ball movement tracking and player positioning on a virtual pitch representation
* **Collaborative Event Logging:** Multiple trackers can work simultaneously on the same match, with real-time synchronization of all recorded events
* **Piano Input Interface:** Specialized keyboard-based input system for rapid event recording during live matches

### Video Analysis Integration
* **YouTube Video Analysis:** Integrated backend function for analyzing YouTube match footage and extracting insights
* **Video Synchronization:** Ability to link recorded events with video timestamps for comprehensive analysis
* **Multi-source Video Support:** Support for various video sources and formats for match analysis

### Team & Player Management
* **Complete Team Setup:** Comprehensive team and player roster management with detailed player profiles
* **Formation Management:** Advanced tactical formation setup and visualization tools
* **Player Assignment System:** Detailed player role and position assignment capabilities

### Advanced Analytics & Visualization
* **Interactive Dashboards:** Rich dashboards with key performance indicators and real-time metrics
* **Statistical Analysis:** Comprehensive statistical tables and performance metrics
* **Advanced Charting:** Multiple chart types including radar charts, heatmaps, and timeline visualizations
* **Player Heatmaps:** Detailed player movement and activity visualization on the pitch
* **Team Performance Analytics:** Comprehensive team-level statistical analysis and comparison tools

### Collaborative Features
* **Real-time Multi-user Collaboration:** Multiple analysts can work on the same match simultaneously
* **Event Type Assignments:** Administrators can assign specific event types to individual trackers for specialized analysis
* **Voice Collaboration:** Integrated voice communication system for coordinating between team members
* **Tracker Activity Monitoring:** Real-time monitoring of tracker activity and availability

### Administrative Tools
* **User Management:** Comprehensive user administration with role-based access control (Admin, Manager, Tracker, Teacher, User)
* **Match Management:** Full match lifecycle management from creation to analysis
* **Event Assignments:** Administrative tools for assigning specific event types to trackers
* **Tracker Absence Management:** Automated systems for managing tracker availability and finding replacements
* **Battery Monitoring:** Real-time monitoring of tracker device battery levels with automated notifications
* **Audit Logs:** Comprehensive logging and auditing of all system activities
* **Notification System:** Advanced notification system for match reminders, assignments, and system alerts

### Advanced Tracking Features
* **Specialized Tracker Assignment:** Ability to assign trackers to specific players or event types for focused analysis
* **Replacement Tracker System:** Automated system for finding and assigning replacement trackers when needed
* **Activity Monitoring:** Real-time tracking of user activity and engagement during matches
* **Device Status Monitoring:** Comprehensive monitoring of tracker device status and connectivity

### Authentication & Security
* **Role-based Access Control:** Multiple user roles with appropriate permissions and access levels
* **Secure Authentication:** Integrated with Supabase authentication for secure user management
* **Profile Management:** Comprehensive user profile system with customizable settings

## Technology Stack

The application is built using a modern, scalable web technology stack:

* **Frontend:** React 18, TypeScript, Vite (for fast development and building)
* **Styling:** Tailwind CSS with shadcn/ui component library for consistent, professional UI
* **State Management:** TanStack React Query for efficient data fetching and caching
* **Backend:** Supabase (PostgreSQL database, real-time subscriptions, authentication)
* **Real-time Features:** Supabase real-time subscriptions for live collaboration
* **Serverless Functions:** Supabase Edge Functions for advanced backend processing
* **Icons:** Lucide React for consistent iconography
* **Charts & Visualization:** Recharts library for data visualization
* **Mobile Support:** Capacitor for potential mobile app deployment

## Database Architecture

The application uses a sophisticated database schema including:
* User profiles and role management
* Match data and event tracking
* Tracker assignments and activity monitoring
* Notification systems
* Real-time collaboration features
* Device status and battery monitoring

## Target Users

This application is designed for:

* **Football Coaches:** Analyze team performance, opponent strategies, and individual player development
* **Sports Analysts:** Conduct in-depth statistical analysis and generate comprehensive reports
* **Football Organizations:** Manage multiple teams, matches, and analyst workflows
* **Data Scientists:** Access rich football data for research and advanced analytics
* **Educational Institutions:** Teaching tool for sports science and analysis programs
* **Professional Clubs:** Enterprise-level match analysis and scouting capabilities

## Use Cases

* **Live Match Analysis:** Real-time event tracking during live matches with multiple analysts
* **Post-Match Review:** Comprehensive analysis of completed matches with video integration
* **Team Performance Monitoring:** Long-term tracking of team and player performance trends
* **Opponent Scouting:** Detailed analysis of opponent teams and player tendencies
* **Training Session Analysis:** Application of analysis tools to training sessions and scrimmages
* **Educational Purposes:** Teaching match analysis concepts and methodologies

The platform serves as a complete solution for modern football analysis, combining traditional statistical tracking with advanced collaborative features and administrative tools for professional-grade match analysis workflows.

**1. القيمة المقترحة (مخصصة للجزائر):**

*   **للأندية المحترفة (الرابطة المحترفة الأولى والثانية):**
    *   اكتساب ميزة تنافسية من خلال رؤى تكتيكية قائمة على البيانات وتحليل المنافسين.
    *   تحسين أداء اللاعبين وتطويرهم وعمليات انتقاء المواهب.
    *   تعزيز كفاءة التدريب من خلال تبسيط سير العمل وأدوات التعاون.
    *   إضفاء الطابع الاحترافي على عمليات النادي والتحضير للمباريات.
*   **لأندية الهواة والأكاديميات (بطولة ما بين الرابطات، بطولات الشباب):**
    *   الوصول إلى أدوات تحليل احترافية بتكلفة معقولة لم تكن متاحة سابقًا.
    *   تحسين جودة التدريب وتطوير اللاعبين من القاعدة.
    *   تحديد ورعاية المواهب المحلية بشكل أكثر فعالية.
*   **للاتحادية الجزائرية لكرة القدم (FAF) والمنتخبات الوطنية:**
    *   توحيد معايير جمع البيانات وتحليلها عبر المنتخبات الوطنية.
    *   مراقبة أداء اللاعبين في الدوريات المحلية لاكتشاف المواهب للمنتخب الوطني.
    *   دعم المديرين الفنيين ومدربي المنتخبات الوطنية بتحليلات متقدمة.
*   **لوسائل الإعلام والمحللين الأفراد:**
    *   توفير بيانات غنية ودقيقة لإنشاء المحتوى والبث والتحليل المستقل.

**نقاط البيع الرئيسية للسوق الجزائري:**
*   **منصة باللغة الفرنسية:** ميزة كبيرة كونها لغة شائعة الاستخدام في الأوساط الرياضية. (التفكير في التعريب الكامل على المدى الطويل لجاذبية أوسع).
*   **حل شامل:** منصة متكاملة تقلل الحاجة إلى أدوات متعددة ومتفرقة.
*   **قابلية التكيف:** تخدم مختلف مستويات كرة القدم، من المحترفين إلى الهواة.
*   **التركيز على التعاون:** تلبي الحاجة إلى العمل الجماعي في التحليل.

**2. شرائح العملاء (حسب الأولوية في الجزائر):**

*   **الأهداف الأساسية:**
    *   **أندية الرابطة المحترفة الأولى والثانية:** الأعلى إمكانية للتبني والإيرادات.
    *   **الاتحادية الجزائرية لكرة القدم (FAF):** للمنتخبات الوطنية، المديرية الفنية، وربما لتوحيد المعايير عبر الدوريات.
*   **الأهداف الثانوية:**
    *   **دوريات الهواة الكبرى (مثل بطولة ما بين الرابطات، بطولة قسم الهواة):** الأندية ذات الطموح وبعض الميزانية.
    *   **أكاديميات الشباب البارزة:** (التابعة للأندية أو المستقلة) التي تركز على تطوير المواهب.
    *   **الجامعات/معاهد الرياضة:** للبرامج التعليمية في علوم الرياضة والتحليل.
*   **الأهداف المتخصصة/الفرعية:**
    *   **وسائل الإعلام الرياضية والصحفيون:** للمحتوى القائم على البيانات.
    *   **المحللون/الكشافون المعتمدون الأفراد.**

**3. القنوات:**

*   **فريق مبيعات مباشر:** ضروري للتواصل مع الأندية المحترفة والاتحادية. يتطلب وجودًا محليًا أو شراكات.
*   **التسويق عبر الإنترنت:**
    *   الموقع الإلكتروني (بالشكل الحالي، مع ضمان أدائه الجيد في الجزائر).
    *   التسويق عبر وسائل التواصل الاجتماعي (فيسبوك، إنستغرام، استهداف مجتمعات كرة القدم الجزائرية، بالفرنسية والعربية).
    *   تحسين محركات البحث (SEO/SEM) بالتركيز على الكلمات المفتاحية المتعلقة بتحليل كرة القدم الجزائرية.
*   **الشراكات:**
    *   **الاتحادية الجزائرية (FAF) والرابطة المحترفة لكرة القدم (LFP):** الشراكات الرسمية أو الاعتمادات ستكون ذات تأثير كبير.
    *   **جمعيات/اتحادات المدربين في الجزائر:** تقديم الأداة في اجتماعاتهم/ندواتهم.
    *   **الموزعون/الاستشاريون المحليون في تكنولوجيا الرياضة:** إن وجدوا أو يمكن تطويرهم.
*   **ورش عمل وعروض توضيحية:** إجراء عروض حية وورش تدريب للأندية والمدربين.
*   **تسويق المحتوى:** دراسات حالة (افتراضية أو لأوائل المتبنين) لأندية جزائرية تستفيد من المنصة.

**4. العلاقات مع العملاء:**

*   **إدارة حسابات مخصصة:** لأندية الرابطة المحترفة الأولى والثانية والاتحادية (الباقات الممتازة).
*   **التأهيل والتدريب:** حاسم للتبني. تقديم تدريب عن بعد وربما تدريب ميداني (خاصة للمستويات العليا). توفير دروس تعليمية بالفرنسية.
*   **الدعم عبر الإنترنت:** البريد الإلكتروني، الدردشة، قاعدة المعرفة (الدعم باللغة الفرنسية ضروري).
*   **بناء المجتمع:** منتدى أو مجموعة مستخدمين للمستخدمين الجزائريين لتبادل أفضل الممارسات.
*   **تحديثات منتظمة وجمع الملاحظات:** إظهار الالتزام بتطوير المنصة بناءً على الاحتياجات المحلية.

**5. مصادر الإيرادات (تسعير مقترح بالدينار الجزائري - DZD):**

*   **باقات الاشتراك (شهرية/سنوية):**
    *   **"أساسي الجزائر" (مقتبس من Starter):**
        *   **الفئة المستهدفة:** أندية الهواة الصغيرة، المدربون الأفراد.
        *   **الميزات:** عدد محدود من المباريات (مثلاً 3-5 شهريًا)، محلل أو اثنان، تتبع الأحداث والإحصائيات الأساسية، مزامنة فيديو محدودة.
        *   **مثال للسعر:** 15,000 - 20,000 د.ج/شهريًا.
    *   **"أداء الجزائر" (مقتبس من Professional):**
        *   **الفئة المستهدفة:** أندية الرابطة الثانية، أندية الهواة الكبرى، الأكاديميات الكبيرة.
        *   **الميزات:** عدد أكبر من المباريات (مثلاً 10-15 شهريًا)، 5-8 محللين، إحصائيات متقدمة (خرائط حرارية، رادار)، مزامنة فيديو كاملة، إدارة الفرق.
        *   **مثال للسعر:** 45,000 - 60,000 د.ج/شهريًا. (تقديم خصم للدفع السنوي).
    *   **"نخبة الجزائر" (مقتبس من Enterprise):**
        *   **الفئة المستهدفة:** أندية الرابطة الأولى، الاتحادية الجزائرية.
        *   **الميزات:** مباريات غير محدودة، محللون غير محدودين، جميع الميزات بما في ذلك أدوات الإدارة المتقدمة، التعاون الصوتي، واجهة برمجة تطبيقات (API)، دعم ذو أولوية، تأهيل مخصص.
        *   **مثال للسعر:** حسب الطلب (يتم التفاوض عليه بناءً على الاحتياجات المحددة، عدد الفرق، التخصيص). من المحتمل أن يبدأ من 150,000 د.ج/شهريًا فما فوق.
*   **خدمات إضافية:**
    *   **دعم متميز وتدريب مخصص:** للأندية التي تحتاج إلى مساعدة مكثفة.
    *   **إنشاء تقارير مخصصة/استشارات:** للأندية التي ترغب في تحليلات أعمق أو تحليل محدد للمنافسين.
    *   **خدمات نقل البيانات:** إذا كانت الأندية تنتقل من نظام آخر.
    *   **برامج الاعتماد/الشهادات:** للمحللين الراغبين في الحصول على اعتماد لاستخدام منصتك.
*   **إيرادات مستقبلية محتملة:**
    *   **بيع البيانات المجمعة والمجهولة المصدر:** لوسائل الإعلام أو البحث (مع موافقة واضحة واعتبارات الخصوصية).
    *   **رعاية/شراكات لميزات أو أحداث معينة.**

**6. الأنشطة الرئيسية:**

*   **توطين المنصة (إذا تم التوسع إلى اللغة العربية) والصيانة.**
*   **المبيعات والتسويق التي تركز على السوق الجزائري.**
*   **تأهيل العملاء وتدريبهم ودعمهم (بالفرنسية، وربما العربية).**
*   **بناء علاقات مع أصحاب المصلحة الرئيسيين في كرة القدم الجزائرية (الاتحادية، الرابطة، رؤساء الأندية، المديرون الفنيون).**
*   **فهم طرق الدفع/البنية التحتية المحلية والتكيف معها.**
*   **ضمان أمن البيانات والامتثال لأي لوائح محلية.**

**7. الموارد الرئيسية:**

*   **المنصة البرمجية نفسها.**
*   **فريق تطوير ماهر (يمكن أن يكون عن بعد).**
*   **فريق مبيعات ودعم محلي/شركاء محليون (حاسم للتواجد على الأرض).**
*   **معرفة بالنظام البيئي لكرة القدم الجزائرية.**
*   **رأس مال مالي لدخول السوق والعمليات الأولية.**

**8. الشراكات الرئيسية:**

*   **الاتحادية الجزائرية (FAF) والرابطة المحترفة لكرة القدم (LFP) (كما ذكرنا):** للمصداقية والوصول والتفويضات المحتملة.
*   **البنوك المحلية/مزودي بوابات الدفع:** لتسهيل المعاملات بالعملة المحلية.
*   **موردي المعدات الرياضية أو مقدمي برامج تعليم التدريب:** فرص ترويج مشترك.
*   **الجامعات ذات البرامج الرياضية:** دمج الأداة في المناهج الدراسية.
*   **شركات الاتصالات:** لإمكانية التجميع أو ضمان اتصال جيد للمستخدمين.

**9. هيكل التكاليف:**

*   **تطوير البرمجيات والبحث والتطوير (مستمر).**
*   **الاستضافة السحابية والبنية التحتية (النظر في الاستضافة المحلية إذا كانت سرعة الاستجابة مشكلة أو لسيادة البيانات).**
*   **الرواتب: موظفو المبيعات والدعم والتسويق المحليون. مطورون عن بعد.**
*   **نفقات التسويق والإعلان (مخصصة للجزائر).**
*   **السفر وتطوير الأعمال (لاجتماع الأندية، الاتحادية).**
*   **التكاليف الإدارية والقانونية (تسجيل الشركة إذا تم إنشاء كيان محلي).**
*   **رسوم معالجة الدفع.**

**استراتيجية دخول السوق واعتبارات للسوق الجزائري:**

1.  **برنامج تجريبي:** قدم المنصة لنادٍ أو ناديين من الرابطة الأولى واثنين من أندية الرابطة الثانية الطموحة بسعر مخفض بشكل كبير (أو مجانًا لفترة محدودة) مقابل الحصول على ملاحظات وشهادات. هذا يبني دراسات حالة.
2.  **استهداف الاتحادية الجزائرية مبكرًا:** الحصول على موافقتهم أو تبنيهم للمنتخبات الوطنية يمكن أن يخلق دفعة من الأعلى إلى الأسفل.
3.  **التواجد المحلي:** ضع في اعتبارك مكتبًا محليًا صغيرًا أو ممثلًا/شريكًا محليًا مخصصًا. هذا يبني الثقة ويسهل العلاقات.
4.  **حساسية الأسعار:** ابحث عن متوسط ميزانيات الأندية. أمثلة الدينار الجزائري توضيحية؛ التسعير الحقيقي يحتاج إلى التحقق من السوق.
5.  **طرق الدفع:** تأكد من قدرتك على قبول طرق الدفع المحلية. قد يكون انتشار البطاقات عبر الإنترنت أقل مما هو عليه في أوروبا. قد تكون التحويلات المصرفية أو حتى النقد (عبر شريك محلي) ضرورية في البداية لبعض الشرائح.
6.  **البنية التحتية:** كن على دراية بمشاكل الاتصال بالإنترنت، خاصة خارج المدن الكبرى. القدرات غير المتصلة بالإنترنت أو المزامنة الفعالة للبيانات لنظام "إدخال البيانو" قيّمة.
7.  **بناء الثقة:** أكد على أمن البيانات وملكيتها، وهي مخاوف عالمية.
